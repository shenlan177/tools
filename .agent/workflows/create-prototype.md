# 创建产品原型页面

使用此工作流让 AI 帮你创建产品交互原型。

## 规范要求

- ✅ **单文件原则**：完整的交互原型在一个 HTML 文件中
- ✅ **高保真设计**：接近真实产品的视觉效果
- ✅ **完整交互**：按钮、表单、弹窗等交互可操作
- ✅ **模拟数据**：使用静态 JSON 模拟真实数据
- ✅ **注释模式**（可选）：点击模块显示功能说明

## 原型存放位置

- **开发中**：`/Users/luckincoffee/antigravitycode/<项目名>/`
- **成熟版本**：复制到 `github-tools/pages/prototypes/`

## 使用方法

直接告诉 AI：
> "帮我创建一个 [产品功能] 的交互原型，包含 [具体模块]"
